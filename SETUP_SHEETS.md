## 권한 설정

1. 스프레드시트에서 "공유" 클릭
2. 서비스 계정 이메일 추가: `signature-dashboard-service@trigger-dashboard-457700.iam.gserviceaccount.com`
3. 권한: "편집자"로 설정
4. "전송" 클릭

## 테스트 방법

1. 회원가입: http://localhost:3001/register
2. 스프레드시트에서 새 행이 추가되는지 확인
3. 로그인: http://localhost:3001/login  
4. 대시보드 접속 확인
